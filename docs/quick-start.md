# 快速开始

1. 先看 `rules/core.md`。
2. 再看你要用的技术栈规则：
   - Java 后端看 `rules/backend-java.md`
   - Ant Design Pro 前端看 `rules/frontend-ant-design-pro.md`
3. 在真实项目根目录放自己的 `AGENTS.md`。
4. 把 `ai-dev-rules` 作为通用规则参考。
5. 开始改代码前，先按规则顺序读一遍。

## 最短接入法

- 通用规则：`ai-dev-rules/rules/core.md`
- 技术栈规则：对应 `rules/*.md`
- 项目规则：当前项目的 `AGENTS.md`

## 只要记住一件事

项目内规则优先，通用规则兜底。
