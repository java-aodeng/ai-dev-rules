# Project Naming Rules

## Current Focus

- 默认主要开发目标是当前主项目及其管理端、H5 和伙伴端应用。
- 其他产品默认只作为参考代码来源，不主动扩展为当前主开发范围；只有用户明确点名时，才切到对应项目。

## Product Mapping

| Product | Backend | Frontend | Notes |
|---------|---------|----------|-------|
| 主产品 A | `backend-service-a` | `admin-app-a` | Main project |
| 参考产品 B | `backend-service-b` | `admin-app-b` | Reference project |
| 参考产品 C | `backend-service-c` | `admin-app-c` | Reference project |
